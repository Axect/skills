# Skills Collection

A small curated collection of reusable skills.

## Included skills

- `dropbox` — upload, download, and share files via Dropbox
- `paperbanana` — generate academic diagrams and plots
- `research-log` — manage research-log workflows and references
- `vastai` — manage Vast.ai GPU cloud workflows

## Structure

Each skill lives in its own directory and includes a `SKILL.md` file.
Some skills also include bundled `scripts/` or `references/` directories.

```text
skills/
├── dropbox/
├── paperbanana/
├── research-log/
└── vastai/
```
